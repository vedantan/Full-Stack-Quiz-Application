# Quiz App - Full Stack Microservices

This is a full-stack quiz application built using:

- React (Frontend)
- Spring Boot Microservices
- Eureka Server
- Spring Cloud API Gateway
- MySQL Database

## Architecture

React → API Gateway → Microservices

Services:
- Question Service
- Quiz Service

## How to Run

1. Start Eureka Server
2. Start Question Service
3. Start Quiz Service
4. Start API Gateway
5. Run React frontend
